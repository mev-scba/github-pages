Ayuda Usuario

Por qué no recibo por email las novedades de mis expedientes ?


No recibe por email las novedades de sus expedientes por alguna de estas razones:

1.- Entre sus datos personales como Usuario MEV, usted no ha cargado su dirección de correo electrónico o,
cargó una dirección errónea, inválida o,
su proveedor de internet momentáneamente no le está prestando servicio.
2.- No agregó el expediente a un Set de Búsqueda
Para incorporar causas a un Set de Búsqueda usted debe realizar cualquiera de la consultas tradicionales (por carátula, n° de causa o n° de receptoría)
y una vez que está viendo la causa, va a poder incorporarla a alguno de los Sets que ya tenga creados, marcando las causas deseadas
y seleccionando el Set de Búsqueda en la parte superior.
3.- o, Incorporá la causa a un Set de Búsqueda, pero el Set NO tiene TILDADO el ítem *Quiere recibir avisos por email ?*
Automáticamente se le enviará un email cada vez que recibe un nuevo paso procesal en la/s causa/s que están incluidas en ese set.
4.- o, no tiene ningún Set de Búsqueda creado.
Para crear un Set de Búsqueda debe hacer click en *Nuevo Set*.

OBSERVACION: Las causas que tramitan en fuero familia NO RECIBEN AVISOS por email.

Ayuda Usuario

Preguntas frecuentes

USUARIOS

Definición: Usted puede definirse como usuario de la Mesa de Entradas Virtual.
Para definirse como Usuario debe ingresar los datos personales que el sistema le solicita y debe elegir un nombre de usuario y una contraseña que solo usted sepa, para identificarse al ingresar. Teniendo en cuenta que su nombre de usuario lo va a identificar, el mismo no debe repetirse para diferentes usuarios, por lo que si el sistema no le permite el nombre elegido por usted, debe elegir otro ya que otro usuario lo ha tomado con anterioridad.
Esto le permitirá crear conjuntos de causas (llamados Sets de Búsqueda) para poder consultar rápidamente las causas que sean de su interés. Además podrá recibir por mail las novedades que se produzcan en las causas que integran su Set de Búsqueda.

Creación de Sets de Búsqueda:

Para crear un Set de Búsqueda debe elegír el vínculo que dice "Nuevo Set", allí el sistema le solicitará un nombre para el Set y también le pedirá que indique si quiere recibir avisos por mail de las causas que integren ese Set. Este servicio consiste en que el servidor le enviará un email automático cada vez que recibe un nuevo paso procesal en la causa.
Para incorporar causas al Set lo debe hacer mediante las consultas. Vea el siguiente punto.

Incorporar Causas al Set: Para incorporar causas a un Set de Búsqueda usted debe realizar cualquiera de la consultas tradicionales (por carátula, número de causa o número de receptoría) y una vez que está viendo la causa, va a poder incorporarla a alguno de los Sets que ya tenga creados, marcando las causas deseadas y seleccionando el Set de Búsqueda en la parte superior.

CONSULTAS

Por Carátula: Ingrese las palabras a buscar en la carátula. El sistema buscará las causas que contengan TODAS las palabras ingresadas por usted en cualquier orden.
Por Número de Causa: Ingrese el número de la causa que usted busca. El sistema buscará la causa que tenga exactamente ese número de causa. Deben incluirse las barras (/) y los guiones (-) si es que el número de causa los contiene.
Por Número de Receptoría: Ingrese el número que le asignó la receptoría a la causa que usted busca. El sistema buscará la causa que tenga exactamente ese número dado por la receptoría. Deben incluirse las barras (/) y los guiones (-) si es que el número en receptoría los contiene.
Por Set de Búsqueda: Seleccione el Set de Búsqueda que desea consultar. Podrá seleccionar alguno de los que usted haya definido. El sistema le mostrará la información de las causas que están en su Set, agrupadas por Organismo. Al ver la lista de las causas, si el Set de Búsqueda contiene causas de diferentes organismos, usted podrá seleccionar el Organismo para ver las causas de su Set en ese Organismo.


Avisos por Correo Electrónico


El sistema de mesa virtual ofrece una función de aviso por medio de correo electrónico. La misma se encuentra disponible para todos los que, además de registrarse, definan en "sets" de búsqueda las causas relativas a su labor profesional. Para ello esá previsto que una vez por día, el sistema emita un correo electrónico a su dirección de email con las novedades del día anterior.

El sistema funciona bajo los siguientes requerimientos:
1) Completar la ficha de registración, habilitada en la opción que el sistema presenta como "Usuario Nuevo", ingresando la dirección de correo electrónico. O bien cambiándola, si fuese necesario, en la opción "Modificar Perfil".
2) Crear los sets de búsqueda deseados prestando conformidad para la recepción de avisos por e-mail.
3) Registrar con precisión en cada set de búsqueda las causas en las que intervenga profesionalmente.

Los avisos cursados por este medio no revisten la calidad de notificación estatuidas por las reglas procesales vigentes, y por el momento sólo constituyen una ayuda para agilizar el trabajo.
